дҪҝз”ЁHE.NETзҡ„Dynamic DNSжңҚеҠЎ

еҸ‘иЎЁдәҺ 2017-11-29 жӣҙж–°дәҺ 2022-09-29 еҲҶзұ»дәҺ Linuxж•ҷзЁӢ
жң¬ж–Үеӯ—ж•°пјҡ 1.9k йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ

дёәд»Җд№Ҳз”ЁHE.NET?

HE.NETжҳҜжҲ‘еңЁ2004е№ҙеүҚеҗҺе°ұжҺҘи§ҰиҝҮзҡ„жңҚеҠЎе•ҶпјҢеҗҺжқҘжҲ‘зҡ„жңҚеҠЎеҷЁйғҪж”ҫеҲ°TheplanetпјҢз„¶еҗҺTheplanetзҡ„дёҖжү№е·ҘзЁӢеёҲзӢ¬з«ӢеҲӣе»әдәҶSoftlayerпјҢеҶҚд№ӢеҗҺSoftlayerиў«IBMж”¶иҙӯпјҢеҶҚеҸҚиҝҮжқҘж”¶иҙӯTheplanetпјҢжҲ‘зҡ„NеҸ°жңҚеҠЎеҷЁз»ҲдәҺйғҪйӣҶдёӯеҲ°дёҖе®¶дәҶгҖӮиҮіе§ӢиҮіз»ҲпјҢе’Ңhe.netдәӨжөҒиҜўд»·иҝҮж— ж•°ж¬ЎпјҢдҪҶжҳҜе§Ӣз»ҲжІЎиғҪеҗҲдҪңгҖӮ

2000е№ҙеүҚеҗҺж— иҒҠеҺ»еҒҡдёҖдёӘDNSе·Ҙе…·пјҢе…¶е®һе°ұжҳҜжү“з®—еңЁWindowsдёҠе®Ңе…Ёд»Һйӣ¶ејҖе§Ӣе®һзҺ°дёҖдёӘеёҰеӣҫеҪўз•Ңйқўзҡ„DNSжҹҘиҜўе·Ҙе…·гҖӮе·ІеҸ‘еёғзүҲжң¬ж”ҜжҢҒдәҶеҫҲеӨҡеёёи§Ғзҡ„DNSи®°еҪ•зұ»еһӢпјҢж–°зүҲжң¬ж”ҜжҢҒдёҖдәӣDNSSEC(Domain Name System Security Extensions)пјҢдҪҶжҳҜеҗҺжқҘеҝҷеҲ«зҡ„дәӢжғ…пјҢе°ұжІЎеҺ»иҖҗеҝғеҺ»еҒҡеҺ»дәҶгҖӮеҒҡзҡ„иҝҮзЁӢиҰҒжөӢиҜ•е•ҠпјҢжҲ‘еңЁDNSжңҚеҠЎеҷЁйҮҢи®ҫзҪ®еҘҪеҜ№еә”зҡ„и®°еҪ•пјҢз„¶еҗҺеҶҚз”ЁжҲ‘е®һзҺ°зҡ„е·Ҙе…·еҺ»жҹҘиҜўпјҢи—үжӯӨеҲӨж–ӯжҲ‘зҡ„е·Ҙе…·е®һзҺ°зҡ„жҳҜеҗҰжӯЈеёёгҖӮ

иҷҪз„¶жҲ‘зҡ„еҮ еҸ°зӢ¬з«ӢжңҚеҠЎеҷЁйғҪиЈ…зқҖDNSжңҚеҠЎпјҢдҪҶжҳҜжүӢеҠЁеҺ»ж”№й…ҚзҪ®ж–Үд»¶еӨӘйә»зғҰпјҢйЈҺйҷ©д№ҹй«ҳпјҢдёҮдёҖжҗһжӯ»дәҶе°ұдјҡжҢЁйӘӮгҖӮдәҺжҳҜжҲ‘е°ұеҺ»жүҫдёҖдәӣDNSжңҚеҠЎе•ҶжөӢиҜ•пјҢжүҫжқҘжүҫеҺ»пјҢеҸ‘зҺ°HE.NETеҜ№еҗ„з§Қи®°еҪ•ж”ҜжҢҒзҡ„жңҖе…Ёйқў*гҖӮдәҺжҳҜе®ғе°ұжҲҗдәҶжҲ‘жөӢиҜ•DNSзҡ„йҰ–йҖүжңҚеҠЎе•ҶгҖӮ

Dynamic DNS жңҚеҠЎе•ҶеҘҪеӨҡе®¶пјҢдҪҶжҳҜж—ўз„¶HE.NETж”ҜжҢҒпјҢжҲ‘е°ұдёҚиҲҚиҝ‘жұӮиҝңе•ҰгҖӮ

DNSе’Ң Dynamic DNS

иҰҒиҜҙDynamic DNSпјҢе°ұдёҚеҫ—дёҚжҸҗ**DNS(еҹҹеҗҚи§ЈжһҗжңҚеҠЎ)**пјӣиҰҒжҸҗDNSпјҢе°ұдёҚеҫ—дёҚжҸҗеҹҹеҗҚгҖӮ

з®ҖеҚ•зҡ„и®ІпјҢ**дә’иҒ”зҪ‘зҡ„дё»жңәйқ IPжқҘеҢәеҲҶпјҢиҰҒи®ҝй—®дёҖеҸ°дё»жңәдёҠзҡ„жңҚеҠЎ(HTTPгҖҒ FTPгҖҒEMAILгҖҒDATABASEзӯү)жҲ‘д»¬йҰ–е…ҲиҰҒзҹҘйҒ“еҜ№ж–№зҡ„IPең°еқҖгҖӮдҪҶжҳҜдә’иҒ”зҪ‘зҡ„дё»жңәеҚғеҚғдёҮпјҢжҠҪиұЎзҡ„IPең°еқҖж №жң¬ж— жі•и®°еҝҶпјҢиҖҢеҹҹеҗҚе°ұжҳҜз»ҷиҝҷдёӘең°еқҖиө·дёҖдёӘеҘҪи®°зҡ„еҗҚеӯ—**гҖӮжүҖд»ҘпјҢжҲ‘д»¬е°ұеҸҜд»ҘйҖҡиҝҮwww.baidu.comиҝҷж ·зҡ„еҹҹеҗҚжқҘи®ҝй—®еҲ°зҷҫеәҰпјҢйҖҡиҝҮwww.taobao.comиҝҷж ·зҡ„еҹҹеҗҚжқҘи®ҝй—®еҲ°ж·ҳе®қгҖӮ

иҖҢд»Һ**еҹҹеҗҚеҲ°IPзҡ„иҝҮзЁӢпјҢе°ұжҳҜеҹҹеҗҚи§ЈжһҗпјҢжҳҜз”ұеҹҹеҗҚи§ЈжһҗжңҚеҠЎеҷЁ(Name Server)жқҘе®ҢжҲҗзҡ„**гҖӮ

иҖҢеҸҰеӨ–дёҖз§ҚеңәжҷҜе°ұжҳҜеҲҶй…Қз»ҷдё»жңәзҡ„IPдёҚж—¶зҡ„еҸҳеҢ–гҖӮ иҝҷж—¶еҰӮжһңдҪҝз”Ёжҷ®йҖҡзҡ„еҹҹеҗҚи§ЈжһҗжңҚеҠЎпјҢжҲ‘д»¬йңҖиҰҒдёҚж—¶зҡ„дҝ®ж”№и§Јжһҗи®°еҪ•пјҢжқҘе°ҶеҹҹеҗҚжҢҮеҗ‘жӣҙж–°еҗҺзҡ„IPгҖӮиҝҷж—¶дјҡжңүдёҖдәӣй—®йўҳеӯҳеңЁпјҢжҜ”еҰӮеҸҳеҢ–йў‘з№ҒпјҢдҝ®ж”№е·ҘдҪңйҮҸеӨ§пјӣеҸҳеҢ–ж—¶жңәдёҚе®ҡпјҢж— жі•зЎ®е®ҡдҪ•ж—¶дҝ®ж”№гҖӮ

Dynamic DNS еҫҲеҘҪзҡ„и§ЈеҶіеҰӮдёҠй—®йўҳпјҢе®ғзҡ„еҹәжң¬еҺҹзҗҶе°ұжҳҜпјҢ**еҪ“дё»жңәе…іиҒ”зҡ„IPеҸҳеҢ–ж—¶пјҢиҮӘеҠЁжӣҙж–°и§Јжһҗи®°еҪ•**гҖӮ

жүҖд»ҘпјҢеұҸи”ҪжҺүдёҖдәӣжҠҖжңҜз»ҶиҠӮпјҢйҖҡдҝ—зҡ„и®Іпјҡ

  • дә’иҒ”зҪ‘дёҠзҡ„дё»жңәйқ IPеҢәеҲҶе’Ңи®ҝй—®гҖӮ
  • еҹҹеҗҚз»ҷIPең°еқҖиө·дәҶдёӘеҘҪи®°зҡ„еҗҚпјҢд»ҺеҹҹеҗҚеҲ°IPзҡ„иҝҮзЁӢе°ұжҳҜеҹҹеҗҚи§ЈжһҗгҖӮ
  • Dynamic DNSиҮӘеҠЁжӣҙж–°и§Јжһҗи®°еҪ•пјҢйҖӮеҗҲе…іиҒ”еҲ°дё»жңәзҡ„IPдёҚж—¶еҸҳеҢ–зҡ„еңәжҷҜгҖӮ

(жң¬е°ҸиҠӮж–Үжң¬жқҘиҮӘжҲ‘д»ҘеүҚеңЁе…¶е®ғзҪ‘з«ҷеҸ‘иЎЁзҡ„еҺҹеҲӣж–Үз« пјҢеҰӮжһңдҪ жүҫеҲ°йӣ·еҗҢзҡ„пјҢжІЎй”ҷпјҢйӮЈе°ұжҳҜжҲ‘еҶҷзҡ„ ?пјү

HE.NETзҡ„Dynamic DNS

е…ідәҺд»–д»¬зҡ„DNSжңҚеҠЎд»ҘеҸҠDynamic DNSзҡ„дҪҝз”Ёз»ҶиҠӮпјҢиҜ·еҸӮиҖғпјҡhttps://dns.he.net/

еӨ§иҮҙжӯҘйӘӨ

жіЁеҶҢд№Ӣзұ»зҡ„йқһеёёз®ҖеҚ•пјҢеӨ§е®¶жҢүжҸҗзӨәж“ҚдҪңеҚіеҸҜпјҢиҝҷйҮҢе°ұдёҚиөҳиҝ°дәҶпјҢеҸӘеҲ—еҮәдёҖдәӣе…ій”®жӯҘйӘӨгҖӮ

  • е°ҶиҰҒи§Јжһҗзҡ„еҹҹеҗҚDNSи®ҫзҪ®дёәHE.NETзҡ„DNS
  • еңЁHE.NETзҡ„DNSйқўжқҝдёӯж·»еҠ еҹҹеҗҚ
  • ж·»еҠ еҜ№еә”зҡ„и§Јжһҗи®°еҪ•пјҲжҜ”еҰӮAи®°еҪ•пјүпјҢе№¶йҖүеҸ–вҖңEnable entry for dynamic dnsвҖқ
  • з”ҹжҲҗз”ЁдәҺDDNSе®ўжҲ·з«Ҝзҡ„еҜҶз Ғ(Generate the key used for dynamic DNS updates)
  • еңЁдё»жңәдёҠдҪҝз”ЁDDNS clientгҖӮ(и®ҫзҪ®е®ҡж—¶д»»еҠЎ)

ж“ҚдҪңе®һдҫӢ

  • е°ҶеҹҹеҗҚDNSи®ҫзҪ®дёәHE.NETзҡ„DNS

  • еңЁHE.NETзҡ„DNSйқўжқҝдёӯж·»еҠ еҹҹеҗҚ

  • ж·»еҠ еҜ№еә”зҡ„и§Јжһҗи®°еҪ•

  • з”ҹжҲҗз”ЁдәҺDDNS clientзҡ„еҜҶз Ғ

  • и®ҫзҪ®DDNS client crontab -e ж·»еҠ еҰӮдёӢд»Јз Ғпјҡ */5 * * * * curl "https://dyn.dns.he.net/nic/update" -d "hostname=iot0.xxxx.com" -d "password=XXXXXX" -k д»Јз ҒйғЁеҲҶеҸҜд»ҘжҹҘйҳ…пјҡВ https://dns.he.net/
    еҘҪе•ҰпјҢеӨ§еҠҹе‘ҠжҲҗе•ҠгҖӮ жғіжөӢиҜ•жҳҜеҗҰжҲҗеҠҹпјҢpingдёҖдёӢеҹҹеҗҚпјҢзңӢзңӢжҳҫзӨәзҡ„IP еҶҚз”ЁдёҖдәӣжҹҘзңӢе…¬зҪ‘IPзҡ„е·Ҙе…·пјҢзңӢдёҖдёӢдёӨиҖ…жҳҜеҗҰдёҖиҮҙе°ұеҸҜд»Ҙе•ҰгҖӮ

жҲ–иҖ…дҪ е·Із»Ҹи®ҫзҪ®еҘҪдәҶNATзҡ„з«ҜеҸЈиҪ¬еҸ‘д»ҘеҸҠејҖеҗҜдәҶеҜ№еә”зҡ„HTTPжңҚеҠЎд№Ӣзұ»зҡ„пјҢйӮЈд№ҲзӣҙжҺҘи®ҝй—®еҹҹеҗҚе°ұеҸҜд»ҘзңӢеҲ°зҪ‘з«ҷе–ҪгҖӮжҳҜдёҚжҳҜйқһеёёз®ҖеҚ•пјҒ

жҖ»з»“

жң¬ж–Үз®ҖеҚ•д»Ӣз»ҚдәҶеҹҹеҗҚгҖҒDNSгҖҒDynamic DNSзҡ„еҺҹзҗҶгҖӮ е№¶д»ҘHE.NETзҡ„DNSжңҚеҠЎдёәдҫӢпјҢдёәiot0.xxxx.comж·»еҠ дәҶдёҖдёӘDynamic DNSзҡ„A RecordгҖӮ йҷҗдәҺзҜҮе№…пјҢеҜ№дёҖдәӣжҠҖжңҜз»ҶиҠӮеҸӘеҒҡйҖҡдҝ—зҡ„и§ЈйҮҠпјҢжғідәҶи§ЈеҜ№еә”зҹҘиҜҶзҡ„пјҢиҜ·еҸӮйҳ…зӣёеә”зҡ„жҠҖжңҜж–ҮжЎЈгҖӮ
