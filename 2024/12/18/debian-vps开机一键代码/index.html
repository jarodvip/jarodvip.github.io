初始

  1. SSH连接主机
  2. 运行脚本 apt update -y && apt install -y curl ，为后面的IP体检和DD系统准备
  3. 运行脚本 bash <(curl -sL IP.Check.Place)， 体检一下IP质量，被墙的IP能换就换；有解锁需求的，也要看一下对应的屏蔽情况

第一步 DD系统【可选】

1.运行一键脚本 官网版一键脚本 curl -sS -O https://kejilion.pro/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh github一键脚本 curl -sS -O https://raw.githubusercontent.com/kejilion/sh/main/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh 2. 重装系统, 必须记住新的密码 注意：

  • 不适合配置太弱的主机，DD容易失联，此类主机建议只更新系统
  • 如果出现失联或者无法自动连接，在后台重启一下机器，然后用新密码进行连接
